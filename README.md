# SpiralJS
Create spiral graphs of time-series data using D3.js

Spiral graphs inspired by [this paper](http://ieg.ifs.tuwien.ac.at/~aigner/teaching/ws06/infovis_ue/papers/spiralgraph_weber01visualizing.pdf).
